## Overview

A comprehensive **solar energy feasibility and optimization case study** developed for the **IIT Mandi Case Competition**. This project analyzes solar energy potential, cost-benefit dynamics, and implementation strategies for solar power systems, providing data-driven recommendations backed by quantitative modeling.

---

## Key Features

- **Feasibility analysis** — Solar irradiance data analysis and energy yield estimation
- **Cost-benefit modeling** — ROI calculations, payback period, and NPV analysis
- **Geographic optimization** — Site-specific solar potential assessment
- **Data-driven insights** — Statistical analysis with visual dashboards
- **Competition-ready** — Structured presentation with actionable recommendations
